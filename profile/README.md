### The problem

Decisions get lost in Slack threads and forgotten docs.
Six months later, nobody remembers why X was chosen.
AI agents produce plausible but shallow work.
Research never reaches implementation.

</td>
<td width="50%">

### Our answer

Every decision becomes a git-tracked artifact.
Quality is scored automatically — not assumed.
Depth calibration forces the right level of rigor.
Evidence with expiration — nothing rots in the dark.

### Quick start

```bash
brew install ForgePlan/tap/forgeplan    # or: curl -fsSL https://forgeplan.dev/install.sh | sh
forgeplan init -y                       # initialize workspace
forgeplan route "your next task"        # get depth + artifact pipeline
forgeplan new prd "Feature Name"        # create artifact → fill → validate → prove → ship
```
