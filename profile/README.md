# ForgePlan

**Engineering decision tools and methodology frameworks for AI-native teams.**

We build open-source tools that turn unstructured engineering work into structured, evidence-backed decisions.

---

</div>

### What we do

| Project | Description |
|---|---|
| **[forgeplan](https://github.com/ForgePlan/forgeplan)** | Engineering decision framework — structured artifacts (PRD, RFC, ADR), quality scoring, evidence tracking, AI-agent integration |
| **[marketplace](https://github.com/ForgePlan/marketplace)** | Plugin ecosystem — methodology skills, development tools, reasoning frameworks |
| **[forgeplan.dev](https://forgeplan.dev)** | Official website and documentation portal |

### Core ideas

- **Every decision leaves a trail** — PRD, RFC, ADR, Evidence as git-tracked markdown
- **Quality is scored, not assumed** — R_eff (weakest-link trust) and F-G-R (formality, granularity, reliability)
- **AI agents are first-class users** — 37 MCP tools, methodology-aware routing, depth calibration
- **Evidence decays honestly** — stale decisions surface on their own, nothing rots in the dark

### Get started

```bash
brew install ForgePlan/tap/forgeplan
forgeplan init -y
forgeplan route "your next task"
```
